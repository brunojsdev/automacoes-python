# 🌌 Hub de Automações Python

Interface centralizada desenvolvida para o gerenciamento e execução de scripts de automação. O projeto atua como um catálogo interativo, facilitando o acesso a rotinas de otimização de fluxo de trabalho desenvolvidas em ambiente Python.

## 🛠️ Especificações Técnicas

* **Front-end:** HTML5 Semântico e CSS3 (Grid e Flexbox).
* **Interatividade:** JavaScript Vanilla para manipulação do DOM.
* **Animações:** API Canvas para renderização de partículas em tempo real (Efeito "Canvas Stars").
* **Estética:** Paleta de cores *Deep Space* de alto contraste (Azul Profundo, Roxo e Amarelo Neon).

## 📁 Módulos de Automação Catalogados

O Hub centraliza os seguintes scripts funcionais:

1.  **Organizador de Diretórios:** Classificação lógica de arquivos por extensão e categoria.
2.  **Backup Automático:** Rotina de compressão e versionamento de arquivos locais.
3.  **Monitor de Status:** Verificação de latência e disponibilidade de serviços web.
4.  **Conversor de Imagens:** Processamento em lote para otimização de ativos visuais.
5.  **Limpador de Temporários:** Script de manutenção para remoção de arquivos obsoletos (>30 dias).
6.  **Extrator de PDF:** Extração de texto de múltiplos documentos para formatos editáveis.

## 🚀 Interface e UX

* **Design Responsivo:** Layout adaptável para resoluções desktop e mobile.
* **Navegação Direta:** Cards interativos com links vinculados aos repositórios específicos no GitHub.
* **Performance:** Carregamento otimizado de ativos e animações fluidas via hardware acceleration.

## 📁 Estrutura do Projeto

* `index.html`: Estrutura semântica e catálogo de cards.
* `style.css`: Estilização baseada em variáveis e sistema de grid.
* `script.js`: Lógica da animação de fundo e interações.
