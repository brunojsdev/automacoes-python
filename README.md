# 🌌 Hub de Automações Python | Productivity Tools

Esta interface centralizada atua como um catálogo interativo e dinâmico para diversos scripts de automação desenvolvidos em **Python**. O objetivo principal é consolidar rotinas de otimização de fluxo de trabalho, oferecendo uma experiência visual moderna e organizada para o gerenciamento de tarefas do dia a dia.

## 🛠️ Stack Tecnológica

- **Interface:** HTML5 Semântico e CSS3 Moderno (Grid, Flexbox e Glassmorphism).
- **Lógica de UI:** JavaScript Vanilla para manipulação dinâmica do DOM.
- **Renderização:** API Canvas para o motor de partículas espaciais em tempo real.
- **Identidade Visual:** Paleta _Deep Space_ de alto contraste, mantendo a consistência visual com os demais projetos do ecossistema.

## 📁 Scripts e Bibliotecas Catalogadas

O hub organiza scripts funcionais que utilizam bibliotecas padrão e externas do Python para resolver problemas reais:

1.  **Organizador de Diretórios:** Classificação automática de arquivos utilizando `os` e `shutil`.
2.  **Backup Automático (Zip):** Rotina de compressão e versionamento com `zipfile` e `datetime`.
3.  **Monitor de Status:** Verificação de disponibilidade de serviços web via `requests`.
4.  **Conversor de Imagens:** Processamento em lote e otimização utilizando a biblioteca `Pillow`.
5.  **Limpador de Temporários:** Manutenção de sistema e remoção de arquivos obsoletos (`time` e `os`).
6.  **Extrator de PDF:** Conversão de documentos para texto editável através do `PyPDF2`.

## 🚀 Destaques de UX

- **Efeito Canvas Stars:** Fundo interativo que reage ao tema e proporciona profundidade sem comprometer a performance.
- **Arquitetura SPA-Like:** Navegação fluida entre os cards de automação.
- **Responsividade Total:** Interface adaptada para dispositivos móveis e desktops de alta resolução.
- **Persistência de Tema:** Sistema de Dark/Light mode integrado com preferência salva no navegador.
